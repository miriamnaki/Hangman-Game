## Summary
This is a game applictaion that lets users guess a random generated word. As the user makes wrong guesses,the hangman images are loaded. When the image is fully loaded, the game will end and defeat sound will play. Otherwise a victoty sound will play when the user wins.Either way,the game is reset and a new word is generated.
## Technology used:

* Javascript
* HTML
* CSS
* JQuery
* DOM Manipulation

## Features:
* Keyboard downpress
* Victory and Defeat sound
* New word reset at the end of the game
* Image load
